# 4Dots Report

Projeto desenvolvido no Hackathon 2015 realizado na Campus Festival, com apoio da SODA Virtual e Governo do Estado. Tivemos o desafio de criar uma ferramenta para analise e visualização dos dados abertos do Governo do Estado, com uma interface simples e amigavel para o usuário final.
Para tanto, utilizamos os dados disponiveis no [Portal da Transparência](http://transparencia.pb.gov.br/ "Title") do Governo do Estado da Paraíba.
